# Gestion stock

Permet de manager un stock (ajout, suppression).

## Install

```
git clone https://github.com/jlguenego/njs-jan-2021.git gestion-stock
cd gestion-stock
npm i
npm run build
npm i -g pm2
pm2 start
```

## Usage

Open a browser and go to `http://localhost:5000`.
